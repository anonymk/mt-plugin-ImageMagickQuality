#ImageMagickQuality

Image::MagickのJPEGの圧縮率を変更するMovable Typeプラグインです。

##インストール

ダウンロードした圧縮ファイルを解凍した後、plugins/ImageMagickQualityをMT_DIR/plugins/にアップロードしてください。

##設定

MT_DIR/mt-config.cgiに"ImageMagickQuality"を設定してください。  
設定値は1〜100です。

例：

    ImageMagickQuality 75

##検証バージョン

Movable Type 5.2.3
